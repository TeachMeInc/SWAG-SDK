# JsonListHelper (Class)

Helper class for converting arrays of JSON data into lists.

## Methods

### static FromJson\<T\>

Converts a JSON string into a list of type `T`.

| Parameter              | Description                                         |
|------------------------|-----------------------------------------------------|
| `string json`          | JSON string containing a list of items.            |
| **Return Type**                | `List<T>`                                           |
